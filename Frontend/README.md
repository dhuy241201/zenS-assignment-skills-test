# Jokee Website
![jokee ui](jokee/src/img/example/jokee-desktop-ui.png)

## Installation
```bash
npm i
```
## Rules
- Click **This is Funny!** if you like joke or **This is not funny** if you don't
- When there is no more joke, web will announce and vote buttons will be disabled
